= Groovy REST Doc

Groovy REST Docs helps you to document REST APis. It helps you generate Asciidoctor snippets to document your API by using an Groovy
DSL.

== Installation

Add a Gradle dependency:

`testCompile 'org.grails:rest-docs:0.1'`
